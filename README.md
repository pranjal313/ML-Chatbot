# Create an AI Chatbot using NLP in Python

## Tasks
- [ ] Classify Consumer Complaints with Natural Language Processing
- [ ] Develop chatbot application for end-user for Complaints Resolution
- [ ] Forward complaints to Resident Engineers where manual intervention required

# Environment
-  Language: Python
-  Replication Environment: Docker or VM

## Chatbot Variants to try: 
-  Chatterbot
-  TensorFlow 2.0
-  PyTorch & PyTorch Lightning
-  Rasa
